Hexo+Github的博客搭建

Node.js的配置

下载

首先登陆Node.js网站下载node.js,并安装在自己指定的路径。

安装完后需要配置变量，首先在nodejs文件夹下新建两个文件夹【node_cache】【node_gloabal】，分别存储缓存文件和全局的模块。

接在cmd命令窗户输入

接着配置环境右键我的电脑，点击属性，在弹出的页面框中选中右下角的环境变量。

双击下面的系统变量的Path，分别将【nodejs】、【node.js\node_global\node_modules】文件夹加入变量值在键盘上按下【win+R】,输入cmd，回车，打开cmd窗口，输入node -v,显示Node.js的版本说明安装成功。输入npm -v，显示npm的版本说明安装成功。

Hexo的配置

Git的配置

首先登陆git网站上下载git，安装成功单击鼠标右键会出现两个新选项【Git GUI here】【Git Bash here】。

Hexo的配置

首先在合适位置右键鼠标，选择【Git Bash here】，调出Terminal,输入npm install -g hexo-cli安装Hexo。安装完后，输入mkdir Blog,新建一个叫Blog的文件夹，接下来的操作都在这个文件夹里面进行，出错可以全部删掉重来。接着继续输入cd Blog进入文件夹，接着使用命令hexo init来初始化Hexo。\

Github的配置

这里已经搭好了Hexo，先进入自己的Github首页，点击自己右上角的加号，选择New repository，在Repository里面为自己的仓库命名为自己名字.github.io，这个就是后面作为博客登陆的地址。
